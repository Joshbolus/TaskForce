# TaskForce

TickBox

Our project will focus on developing a simple to-do list website using Python and HTML. 
TickBox will allow users to track manage and track everyday activities in an organized and efficient manner. 

Languages being used
* Python
* HTML
* Flask

Current Planned Features
* Simple and easy-to-use interface to navigate
* Allow users to add/remove tasks and mark them as complete
* Set up daily reminders for added tasks
* Filter added tasks by multiple options
* Save created to-do lists for easy access later
